v7.1.0 FEATURES (INCLUDED)
---------------------------

Major Feature - Runtime Calibration System:
  • Boot-time calibration mode (hold encoder button during power-on)
  • Serial command interface with 8 commands
    - CAL SHOW: Display current calibration values
    - CAL CV/DUTV/DUTC/SHUNT <value>: Set calibration parameters
    - CAL SAVE: Persist values to ESP32 NVS storage
    - CAL RESET: Restore factory defaults
    - CAL EXIT: Exit calibration mode
  • ESP32 Preferences (NVS) storage - survives firmware updates
  • Case-insensitive commands with backspace and Ctrl-C support
  • Four calibration parameters: dutVcalib, DUTCurrent, shuntVcalib, cvCalFactor
  • Two-stage display: Simple welcome → Full command menu
  • Two exit methods: Button click (continue boot) or CAL EXIT (power cycle)
  • Fan automatically stopped during calibration mode
  • Binary distribution friendly - no development tools needed

Storage Technology:
  • ESP32 NVS partition in "dl_cal" namespace
  • Persists across firmware updates, OTA updates, and reflashing
  • Double precision floating point (15+ significant digits)

Documentation:
  • Added CALIBRATION_GUIDE.md with comprehensive procedures
  • Added data/README.md with user-friendly instructions
  • Updated all comments to reflect Preferences storage

CALIBRATION OVERVIEW
--------------------

Entering Calibration Mode:
  1. Power off ESP32
  2. Press and hold encoder button
  3. Power on ESP32 (keep holding button)
  4. Wait for "CALIBRATION MODE" screen
  5. Release button after screen appears

Serial Commands (9600 baud):
  • Type "CAL" alone to show command menu
  • Example: "CAL CV 1.0606" to set CV calibration factor
  • Must use "CAL SAVE" to persist changes
  • Use "CAL EXIT" to exit and require power cycle
  • Or click button to cancel and continue boot

Most Common Calibrations:
  • cvCalFactor: CV mode trigger point (typically 1.0 to 1.10)
  • dutVcalib: DUT voltage display accuracy (typically 0.99 to 1.01)
  • shuntVcalib: Current display accuracy (typically 2.45 to 2.55)

See CALIBRATION_GUIDE.md in repository for detailed procedures.
